## Task: Add WebSocket Keepalive Pings to Prevent Idle Timeout

### Problem
Deepgram closes idle WebSocket connections after ~30s. We need to keep it alive with periodic pings.

**Location:** Around line 45, with other task fields like `self._sender_task`, add:

```python

```

### Change 2: Add keepalive coroutine

**Location:** Add after `_ensure_websocket_connected()` method (around line 210):

```python
async def _keepalive_loop(self) -> None:
    while self._running and self._is_warmed and not self._is_capturing:
        try:
            if self._websocket is not None and not self._websocket.closed:
                await self._websocket.send(b'')
                print("[DEBUG] Keepalive ping sent")
            await asyncio.sleep(15)
        except Exception as e:
            print(f"[DEBUG] Keepalive error: {e}")
            break
```
