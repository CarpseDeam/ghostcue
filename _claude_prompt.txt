## Task: Fix Overlay Text Selection & Code Formatting

### File: `app/overlay.py`

### Problem
1. QLabel doesn't support proper copy operations (no Ctrl+A, awkward selection)
2. Code blocks from Claude lose indentation - renders as plain text
3. User needs to copy text for job applications

### Requirements

- Change `self._text_label` from `QLabel` to `QTextEdit`
- Set read-only mode: `setReadOnly(True)`
- Enable rich text/HTML rendering
- Remove the `QScrollArea` wrapper - QTextEdit has built-in scrolling

**2. Markdown Code Block Rendering**

- Detects markdown code blocks (triple backticks with optional language)

  - Padding: `padding: 8px`
  - Border radius: `border-radius: 4px`
  - Preserved whitespace: `white-space: pre`
- Escapes HTML entities in code blocks (`<`, `>`, `&`)
- Wraps regular paragraphs in `<p>` tags
- Handles inline code (single backticks) with lighter styling
